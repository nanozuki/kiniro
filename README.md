# Kiniro

Kiniro is a web app for creating color themes in OKLCH color space. It helps you
start from colors you like, generate whole palette with lightness steps, and
export to CSS variables.

Features:

1. Create lightness steps in OKLCH from a base color
2. Support multiple themes and multiple variables per theme
3. Export to CSS variables.
4. Manually adjust each steps and colors
5. WCAG contrast checker
